About Me

I am a passionate and dedicated BSIT graduate from Punjab University, specializing in mobile application development. Throughout my degree, I built a strong foundation in core IT concepts including Object-Orientated Programming, Database Systems, and Computer Networks.

My primary focus is on Android development using Java. I enjoy building functional and user-friendly mobile applications. I recently completed my Final Year Project, 'CircleUp,' a comprehensive Android messaging app developed entirely in Java. This project allowed me to gain hands-on experience in building a full-stack mobile application, utilizing technologies such as Java, Android Studio, Firebase for backend services (Authentication, Realtime Database), and Room Database for offline data persistence. I also implemented end-to-end encryption and integrated push notifications using OneSignal.

Building mobile applications and solving real-world problems through code is what drives me. I am now eager to apply these skills in an entry-level software development role and contribute to a dynamic team.
