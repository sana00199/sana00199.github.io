About Me

I am a passionate and dedicated BSIT graduate from Punjab University, specializing in mobile application development with a strong focus on Android using Java. My academic journey provided a solid foundation in core IT concepts, including Object-Oriented Programming, Database Systems, and Computer Networks.

I thrive on building functional and user-friendly mobile applications that solve real-world problems. My capstone achievement is 'CircleUp,' a comprehensive Android messaging app developed entirely in Java for my Final Year Project. This experience provided me with hands-on expertise in full-stack mobile development, integrating technologies like Firebase (Authentication, Realtime Database), Room Database for robust offline persistence, and implementing critical features like end-to-end encryption and push notifications using OneSignal.

I am now eager to apply these skills in an entry-level software development role and contribute to a dynamic and innovative team.
