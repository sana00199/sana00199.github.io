About Me

I am a passionate and dedicated BSIT graduate from Punjab University, specializing in mobile application development. Throughout my degree, I built a strong foundation in core IT concepts including Object-Oriented Programming, Database Systems, and Computer Networks.

I recently completed my Final Year Project, 'CircleUp,' a comprehensive Android messaging app developed entirely in Java. This project allowed me to gain hands-on experience in building a full-stack mobile application, utilizing technologies such as Java, Android Studio, Firebase for backend services (Authentication, Realtime Database), and Room Database for offline data persistence. Building CircleUp significantly enhanced my practical skills in Android development, database integration (Firebase & Room), implementing security features (like E2EE), and using version control (Git). I am now eager to apply these skills in an entry-level software development role and contribute to a dynamic team.
