Sana's Tic Tac Toe: Online Multiplayer

Play & Connect: Real-time Fun with Friends!

Project Overview

Sana's Tic Tac Toe is a dynamic, web-based online multiplayer game developed using HTML, CSS, and JavaScript, leveraging Firebase for real-time data synchronization. This project showcases the ability to build interactive web applications with live updates and user-to-user connectivity, demonstrating foundational skills in web development and backend integration for real-time experiences.

The game allows two players to connect using a unique game ID, providing a seamless and engaging experience across different web browsers and devices.

Key Features & Functionalities

Online Multiplayer Gameplay

  • Supports two-player online mode, allowing friends to play together remotely.
  • **Game ID System:** Players can either start a new game (generating a unique ID) or join an existing game by entering a shared ID.
  • **Easy Sharing:** Game IDs are prominently displayed with a "Copy ID" button for effortless sharing with opponents.

Real-time Synchronization

  • **Live Board Updates:** Game moves (placing 'X' or 'O') are synchronized in real-time across both players' devices using Firebase.
  • **Turn Management:** The game clearly indicates the current player's turn ("It's Your Turn" / "[Opponent's Name]'s Turn"), providing clear guidance.

Game Logic & Session Management

  • **Win/Draw Detection:** Accurately identifies and announces the winner or a draw based on Tic Tac Toe rules.
  • **Replay Options:** After a game concludes, players can choose to "Play Again" (re-match with the same opponent) or "New Game" (start a fresh session).
  • **Session Handling:** Manages game session lifecycle, including notifications for session end.

Clean & Engaging UI/UX

  • Designed with a modern, clean interface featuring a vibrant purple-to-pink gradient background and smoothly rounded cards and buttons.
  • Utilizes a clear and consistent font throughout for enhanced readability.

Challenges & Key Learnings

Developing a real-time online game presented unique and insightful challenges:

Real-time State Synchronization

Problem: Ensuring the game board state, player turns, and winner status were instantly and consistently updated across two different clients over the internet.

Solution: Leveraged Firebase Realtime Database to continuously synchronize game data. Implemented listeners to update the UI immediately upon changes in the database, ensuring a fluid and seamless multiplayer experience.

Game Logic & Win Conditions

Problem: Accurately implementing the Tic Tac Toe win-checking algorithm and managing player turns, especially in an asynchronous, distributed environment.

Solution: Developed a robust JavaScript game engine that managed board state, validated moves, and efficiently checked for winning combinations and draw conditions after every turn, ensuring fair and accurate gameplay.

Session Management & Connectivity

Problem: Creating a simple yet effective system for two players to connect, join a specific game instance, and handle session lifecycles (start, end, rematch).

Solution: Implemented a **unique Game ID system** where Firebase nodes represented individual game sessions. This allowed players to easily share an ID and connect to the same game, facilitating smooth game initiation and re-connection.
