Simple Advanced Calculator

Mastering Arithmetic & Logic on Android

Project Overview

Developed a robust and user-friendly Calculator application for Android using Java. This project served as a foundational exercise in mastering core Android UI concepts, event handling, and implementing complex mathematical logic. It demonstrates proficiency in crafting clean, functional, and intuitive mobile interfaces.

Key Functionalities

Core Operations

  • Basic Arithmetic: Implemented addition (+), subtraction (-), multiplication (×), and division (÷).
  • Advanced Functions: Supported modulo (%), power (^), square root (√), and sign change (+/-) operations.
  • Decimal Input: Enabled precise decimal point input for accurate calculations.
  • Clear & Backspace: Full 'Clear (C)' and single-character 'Backspace (DEL)' functionalities for input correction.

Usability & Logic

  • Error Handling: Robustly managed potential errors such as division by zero and square root of negative numbers.
  • Expression History Display: Continuously displayed the full mathematical expression as it's entered, enhancing user clarity.
  • Repeat Last Operation: Allowed users to repeat the last performed operation by pressing "=" again after a result is shown.
  • Clean & Intuitive UI: Designed a simple, clean, and responsive user interface for ease of use.
