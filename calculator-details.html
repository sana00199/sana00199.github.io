Simple Advanced Calculator

Developed a functional and user-friendly calculator application for Android using Java. This project demonstrates the implementation of core Android UI concepts, event handling, and mathematical logic.

Features:

  • Basic arithmetic operations: Addition (+), Subtraction (-), Multiplication (×), Division (÷)
  • Advanced operations: Modulo (%), Power (^), Square Root (√), Sign Change (+/-)
  • Decimal point input
  • Clear (C) and Backspace (DEL / ←) functionality
  • Handles potential errors like division by zero and square root of negative numbers
  • Displays the full expression history during calculations
  • Allows repeating the last operation by pressing "=" again after a result is shown
  • Simple and clean UI layout

Technologies Used:

  • Java
  • Android Studio
  • XML Layouts (LinearLayout, GridLayout)
  • Android SDK (TextView, Button, View.OnClickListener)
  • Math class (for sqrt, pow)
  • DecimalFormat (for number formatting)
  • Git
  • GitHub

Concepts Applied:

  • Activity Lifecycle (Basic)
  • UI Design and Layout Management
  • Event Handling (Button Clicks)
  • Managing Application State (using variables like currentInput, operand1, operator)
  • String Manipulation and Parsing
  • Basic Arithmetic Logic and Error Handling
  • Implementation of Mathematical Functions
  • History Tracking (using StringBuilder)
  • Handling User Input edge cases (e.g., decimals, leading zeros)
